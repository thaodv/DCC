  Eco Value Acquisition Rules

Eco-values are counted at 0 o'clock on the following day. After calculations are completed, users will be automatically rewarded DCC based on the eco-values. The size of the reward = the single user's ecological contribution value / the total user's ecological contribution value for the day * the daily award amount.

The daily total reward amount is 10,000 DCC per day starting from July 1, 2018. If the number of authenticated users totals 100,000 or 5 merchants, the daily award amount will increase to 270,000 DCC. The reward will come from the Eco Foundation's awarding address to the user's wallet address automatically, without any user
